# Pricing — 가격 책정 유틸리티

가격 모델을 시뮬레이션하고 손익분기점을 계산합니다.

## 실행 순서

1. **프로젝트 비용 구조 파악**
   - 인프라 비용 (호스팅, DB, CDN)
   - 외부 API 비용 (결제, 이메일, AI 등)
   - 고정 비용 vs 변동 비용 구분
2. **가격 모델 3가지 시뮬레이션**

| 모델 | 구조 | 적합 조건 |
|------|------|---------|
| Freemium | 무료 + 유료 업그레이드 | PLG, 낮은 전환 마찰 |
| Subscription | 월/연 정기 구독 | 예측 가능 수익 |
| Usage-based | 사용량 비례 과금 | 가치 비례 전달 |

3. **경쟁사 가격 참조** (수동 입력 기반)
4. **BEP(손익분기점) 계산**
   ```
   BEP = 총 비용 / (ARPU × Gross Margin)
   ```
5. **권장 가격대 출력** — 3가지 모델별 추천 가격 + 근거

> CFO의 pricing-modeler 에이전트 축약 버전. 상세 분석은 `/vais cfo {feature}` 사용.
