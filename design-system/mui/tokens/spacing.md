# Spacing Tokens (mui v1.0.1)

**Unit**: 8px
**Scale**: `0, 4, 8, 16, 24, 32, 48, 64, 96, 128`

## 사용 가이드

- `theme.spacing(n)` = `n * 8` px
- 자주 쓰는 값: 4 (XS), 8 (S), 16 (M), 24 (L), 32 (XL), 48 (XXL)
- 컴포넌트 사이 간격, 패딩, 마진 모두 spacing 토큰만 사용

| Step | Value | Use case |
|:----:|:-----:|----------|
| 0 | 0 | none |
| 0.5 | 4px | inline 미세 |
| 1 | 8px | tight |
| 2 | 16px | default group |
| 3 | 24px | section |
| 4 | 32px | block |
| 6 | 48px | large block |
| 8 | 64px | page |
